# URL para compartir por WhatsApp

## Dónde está tu URL

La **URL para compartir** es la dirección donde tengas publicada la app:

- **Si usas Streamlit Community Cloud** (gratis):  
  1. Sube este proyecto a GitHub.  
  2. Entra en [share.streamlit.io](https://share.streamlit.io), conecta el repositorio y despliega.  
  3. Streamlit te dará una URL como:  
     **`https://[nombre-de-tu-app].streamlit.app`**  
  Esa es la URL que debes enviar por WhatsApp.

- **Si usas otro servidor** (tu propio dominio, VPS, etc.):  
  La URL será la que tú configures (por ejemplo `https://tesoreria.tudominio.com`).

## Qué hacer con la URL

1. Copia la URL de tu app (la que te da Streamlit o tu servidor).  
2. Envíala por WhatsApp a quien quieras que use la app.  
3. Quien la abra en el celular puede instalarla como app: menú del navegador → **"Añadir a la pantalla de inicio"** o **"Instalar aplicación"**.

No hay una URL fija que funcione para todos: cada despliegue tiene su propia URL.
